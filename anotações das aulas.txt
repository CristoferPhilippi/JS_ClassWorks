Desafio de fixação — IMC

Seu material já possui o início desse cálculo:

let altura = 1.8;
let peso = 97;

let resultado = peso / (altura * altura);

Transforme-o em um programa completo:

const peso = 97;
const altura = 1.8;

const imc = peso / (altura * altura);

Depois classifique:

IMC abaixo de 18,5  → Abaixo do peso
IMC abaixo de 25    → Peso normal
IMC abaixo de 30    → Sobrepeso
IMC igual ou maior  → Obesidade

Estrutura inicial:

if (/* condição */) {
  console.log("Abaixo do peso");
} else if (/* condição */) {
  console.log("Peso normal");
} else if (/* condição */) {
  console.log("Sobrepeso");
} else {
  console.log("Obesidade");
}
